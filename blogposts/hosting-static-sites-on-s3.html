Hosting static sites on S3

Turns out this was pretty straightforward. The steps are basically:

  1. decide on a dns name for your static page
  2. create an S3 bucket using that name (bullshit.jowj.net)
  3. upload at least one file to your bucket
  4. change bucket to enable hosting static pages (ensure you set the file you uploaded to be your index)
  5. copy fully qualified aws name of endpoint and point cname (bullshit) to that endpoint
  6. boom done.

I'd had this as a "thing to do" for forever, but I guess I thought it'd be more involved. this was just really straight forward.
